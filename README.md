## Installation

### Local

1. Clone the repository
2. Run `npm install`
3. Run the app with `npm start`

### Deployment

1. Create a production build with `npm run build`
2. For Node based environments:

- Using serve:

```
npm install -g serve
serve -s build
```

- Using Express:

```
npm install express
node server.js
```
